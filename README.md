# FileCherry Landing Page

Landing page for FileCherry - "Files in, cherries out."

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Structure

- `src/app/page.tsx` - Main landing page with all sections
- `src/components/CodyMascot.tsx` - Cody the Cherry Picker mascot component
- `public/cody.png` - Cody mascot image
- `src/app/globals.css` - Global Tailwind styles

## Build

```bash
npm run build
npm start
```
